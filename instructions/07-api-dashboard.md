## API Endpoints

### GET /health
```json
Response 200:
{
  "status": "healthy",
  "model_loaded": true,
  "model_name": "XGBClassifier",
  "version": "1.0.0"
}
```

### GET /
Returns `dashboard.html` with:
- Overall attrition rate from dataset
- Model ROC-AUC score
- Top 3 risk factors (global SHAP)
- Links to predict and report pages

### POST /predict
```json
Request body (example):
{
  "Age": 35,
  "Department": "Sales",
  "JobRole": "Sales Executive",
  "MonthlyIncome": 3500,
  "OverTime": "Yes",
  "YearsAtCompany": 2,
  "JobSatisfaction": 2,
  "WorkLifeBalance": 1,
  "DistanceFromHome": 25,
  "TotalWorkingYears": 8,
  "MaritalStatus": "Single",
  "BusinessTravel": "Travel_Frequently"
}

Response 200:
{
  "attrition_probability": 0.78,
  "risk_level": "HIGH",
  "risk_color": "#EF4444",
  "top_risk_factors": [
    {"feature": "OverTime", "value": "Yes", "impact": 0.34, "direction": "increases_risk"},
    {"feature": "JobSatisfaction", "value": 2, "impact": 0.22, "direction": "increases_risk"}
  ],
  "recommendation": "Immediate HR intervention recommended."
}
```

### GET /report
Returns `report.html` displaying all 8 EDA charts as inline images.

## Pydantic Schema

```python
from pydantic import BaseModel, Field

class EmployeeInput(BaseModel):
    Age: int = Field(..., ge=18, le=70)
    Department: str
    JobRole: str
    MonthlyIncome: float = Field(..., gt=0)
    OverTime: str = Field(..., pattern="^(Yes|No)$")
    YearsAtCompany: int = Field(..., ge=0)
    JobSatisfaction: int = Field(..., ge=1, le=4)
    WorkLifeBalance: int = Field(..., ge=1, le=4)
    DistanceFromHome: int = Field(..., ge=1)
    TotalWorkingYears: int = Field(..., ge=0)
    MaritalStatus: str
    BusinessTravel: str

class PredictionResponse(BaseModel):
    attrition_probability: float
    risk_level: str          # LOW / MEDIUM / HIGH
    risk_color: str          # hex color for UI
    top_risk_factors: list[dict]
    recommendation: str
```

## Risk Level Thresholds

```python
def get_risk_level(probability: float) -> tuple[str, str]:
    if probability >= 0.65:
        return "HIGH", "#EF4444"
    elif probability >= 0.35:
        return "MEDIUM", "#F59E0B"
    else:
        return "LOW", "#10B981"
```
